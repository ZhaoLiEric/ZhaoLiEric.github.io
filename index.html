Biquadris


The game of Biquadris is a Latinization of the well-known game Tetris, consisting of two players competition and seven different types of blocks (with one additional star block). A player obtain scores for filling all the cells of one line, as well as deleting all the cells of one block, where different levels significantly impact the amount of score. One player’s game is over when a new block cannot be properly fitted on the game board. During a player’s turn, the block that the opponent will have to play next is already at the top of the opponent’s board and if it does not fit, the opponent has lost. Player can choose to start a new game or quit when a game is over.

Real-time Rumour Analysis with PySpark


I built a real-time rumour analysis pipeline for tweets in PySpark, which utilizes Spark SQL, Spark ML and Spark Structured Streaming. We use a publicly available dataset of tweets that have been annotated for subtasks related to rumour analysis. This dataset is provided by the organizer of RumourEval 2019 competition, which consists of two subtasks, namely stance detection and rumour veracity classification. We train a traditional machine learning model using manually extracted features for the two tasks separately and compare our results to existing deep learning approaches. We then use one of our models to analyze tweets being posted in real time using the PySpark Structured Streaming API. While most of the current research being done in NLP is focused on deep learning (which primarily uses GPU clusters), the sheer volume of social media posts means there is still value in training traditional machine learning models that can be run on massively parallel CPU clusters.View the code on GitHub.

Movie Review Prediction System


Recently, graphical neural networks (GNN) has attracts great attention, where a variety of NLP problems can be best expressed with a graph structure. For instance, the sentence structural information in text sequence can be represented by syntactic parsing trees. However, the application of GNN techniques on the problem of text regression has not been well studied. In this project, the movie review sentiment prediction task is used as a downstream task to study the performance of different GNNs techniques, and a GNN-based movie review prediction system is implemented, which predicts a numerical rating score ranging from 0 to 1 given a movie review text. By comparing different multiple variants of the graph construction, graph representation learning and node embedding initialization, a movie review prediction system with high accuracy is proposed.
